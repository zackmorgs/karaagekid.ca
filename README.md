# karaagekid.ca

## Firebase Hosting

1. Update `.firebaserc` with your real Firebase project ID.
2. Log in once with `npm run firebase:login`.
3. Build and deploy with `npm run deploy`.
4. If you need to connect the project interactively, run `npm run firebase:init`.

## Menu

### Karaage Plates
- **3-Piece Karaage**  
  Crispy Japanese-style fried chicken served over a bed of purple rice

### Karaage Boxes
- **Japanese Curry Box**  
  Rich Japanese curry served with assorted fried vegetables and karaage  

- **Spicy Sesame Box**  
  Karaage with sesame sauce and spicy mayo  

- **Citrus Garlic Box**  
  Karaage with citrus soy glaze and garlic mayo  

### Karaage Sandwiches
Served on a brioche bun with coleslaw, daikon, or pickles  

- **Karaage Sandwich (Mild)**  
  Topped with sesame sauce, spicy mayo, and crispy onions  

- **Nashville Hot Sandwich**  
  Spicy karaage topped with buttermilk ranch and crispy onions  

### Snacks
- **Panko Fried Hot Dog Skewers**  
  Served with sesame sauce and spicy mayo  

- **Takoyaki**  
  Japanese octopus balls served with your choice of sauce
